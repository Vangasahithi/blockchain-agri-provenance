# System Architecture

## Overview

The Blockchain-Based Provenance Tracking and Tamper Detection System is designed using a layered architecture to ensure transparency, traceability, and security throughout the agricultural supply chain.

---

## Architecture Layers

### 1. User Layer
- Farmers
- Distributors
- Retailers
- Consumers

---

### 2. Frontend Layer
- React.js Dashboard
- Product Registration
- Product Tracking
- Ownership Transfer

---

### 3. Backend Layer
- Node.js
- Express.js
- REST APIs

---

### 4. Blockchain Layer
- Hyperledger Fabric
- Smart Contracts (Go Chaincode)
- Ledger

---

### 5. Storage Layer
- IPFS
- Blockchain Ledger

---
