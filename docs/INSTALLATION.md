# Installation Guide

## Prerequisites

- Node.js
- Go 1.21+
- Docker Desktop
- Git
- Hyperledger Fabric Binaries

---

## Clone Repository

```bash
git clone https://github.com/Vangasahithi/blockchain-agri-provenance.git
```

---

## Frontend

```bash
cd frontend
npm install
npm start
```

---

## Backend

```bash
cd backend
npm install
npm start
```

---

## Hyperledger Fabric

Start the Fabric network and deploy the chaincode before running the backend.
